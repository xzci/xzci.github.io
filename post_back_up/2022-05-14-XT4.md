# XT4

```
光圈 在镜头上

右边 快门设置

左边 iso 设置

左边 下面：
模式拨盘：s 单张拍摄模式
        cl 低速连拍
        ch 高速连拍
        bkt 包围曝光模式（有6种模式）
        adv 高级模式（滤镜）
（胶片模拟只能在s，cl， ch 里用，右下按钮左键）

af 是 开启自动对焦
ael 自动曝光

正面 右下
 
m 手动对焦
c 连续自动对焦
s 单词自动对焦

```


## 曝光模式

 - P 挡
  
  光圈（镜头上的） a 

  右面 （快门） a
 
 - A 挡 （光圈优先）
  
  光圈（镜头上的）不是 a 
  
  右面 （快门） a

 - S 挡 （快门优先）街上拍

  光圈（镜头上的） a 
 
  右面 （快门） 不是a

  - M 挡 （自动挡）
  
  光圈（镜头上的）不是 a 
  
  右面 （快门） 不是a

  - 全自动，3个a

  - 3个里面有一个a和曝光补偿一起才会有用，没有a 的话，曝光补偿无用
