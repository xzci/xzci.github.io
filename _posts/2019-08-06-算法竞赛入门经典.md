### 变种一 任意基本变量都交换

传递另外的参数, 使用switch 确定 基本变量的类型.

```c
enum types {
    Int, Double, Float
};

void swap_int(int* a, int* b)
{
    int temp = *a;
    *a = *b;
    *b = temp;
}

void swap_double(double* a, double* b)
{
    double temp = *a;
    *a = *b;
    *b = temp;
}

void swap(void* a, void* b, enum types type)
{
   
    switch (type)
    {
    case Int:
        swap_int((int*)a, (int*)b);
        break;
    case Double:
        swap_double((double*)a, (double*)b);
        break;
    }
}
```

## 3n +１

任意大于1的自然数,若n为奇数, 则将n变为3n + 1, 否则将变为n的一半,经过这样的若干次变换,一定会使n变为1. 例如3 -> 10 -> 5 -> 16 -> 8 -> 4 -> 2 -> 1

输入n,输出变换的次数.

> 3
> 7

```c
int f_1(int n)
{
    int index = 0;
    while (n != 1)
    {
        if (n % 2 == 1)
        {
            n = 3 * n + 1;
        }
        else
        {
            n /= 2;
        }
        index++;
    }
    return index;
}
```

## 近似计算

计算pi/4 = 1 - 1/3 + 1/5 - 1/7 + ...  直到最后一项小于10e-6

```c
double f_1(void)
{
    double sum = 0;
    double item;
    for (int i = 0; ; i++)
    {
        item = 1.0 / (i * 2 + 1);
        if (i % 2)
        {
            sum -= item;
        }
        else
        {
            sum += item;
        }
        if (item < 1e-6)
            break;
    }
    return sum;
    
}
```

## 阶乘之和

输入n, 计算S = 1! + 2! + 3! + .... + n! 的末6位(不含前导0). n <= 1e6

> 10
> 37913

* 只计算末尾6位 (因为只需要末尾的6位,可以只用考虑末尾6位的情况, 防止溢出的可能)
* 全部计算,最后取出6位数据,因为该式为阶乘,所以很大概率会溢出

__特别的__ 25! 末尾有6个0, 所以后面的加和不会影响数据的结果. 
`if(n > 25) n = 25;`

```c
int f_1(int n)
{
    if (n > 25)
        n = 25;
    int sum = 0;
    int temp;
    for (int i = 1; i <= n; i++)
    {
        temp = 1;
        for (int j = 1; j <= i; j++)
        {
            temp = j * temp % 1000000;
        }
        sum = (sum + temp) % 1000000;
        
    }
    return sum;
}
```
