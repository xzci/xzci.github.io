## 圆柱体表面积

输入底面半径r和高h，输出圆柱体的表面积，保留3位小数

> 3.5  9

> Area = 274.889

```C
double f_1(double r, double h) {
    return (r * r * PI) * 2 + ((PI * 2 * r) * h);
}
void f_2(void) {
    double r, h;
    scanf("%lf%lf", &r, &h);
    printf("Area = %.3lf", O_1_1(r, h));
}

// -----------------

void f_simple(void) {
    // one way to get the value of pi
    const double pi = acos(-1.0);
    double r, h;
    scanf("%lf%lf", &r, &h);
    printf("Area = %.3f", (r * r * pi) * 2 + ((pi * 2 * r) * h));
}
```
